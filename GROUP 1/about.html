PROJECT OVERVIEW

Static Website Development

This project is part of the Cloud Computing course (CSRC 252) at Ghana Communication Technology University. Our Group 1 team developed this static website to demonstrate web development and deployment using GitHub Pages as part of the practical lab assignment.


ABOUT THE PROJECT

Objectives

The main objectives of this project include:

  • Understanding cloud computing fundamentals
  • Learning static website development
  • Deploying applications using GitHub Pages
  • Collaborating as a team using version control (Git)
  • Creating responsive and user-friendly web interfaces

Technology Stack

HTML5, CSS3, and JavaScript for front-end development with responsive design principles.
